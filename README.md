# Codesoft Internship Projects  

This repository contains the projects I completed during my **Codesoft Data Science & Machine Learning Internship**. Each project focuses on building 
and evaluating machine learning models while improving data handling and analytical skills.  

---

## 📌 Projects  

### 1. Titanic Survival Prediction  
- **Objective**: Predict whether a passenger survived the Titanic disaster based on features like age, gender, class, etc.  
- **Techniques Used**: Data preprocessing, feature engineering, classification algorithms.  
- **Outcome**: Built a classification model to predict survival probability with improved accuracy.  

---

### 2. Iris Flower Classification  
- **Objective**: Classify iris flowers into three species (*Setosa, Versicolor, Virginica*) using petal and sepal dimensions.  
- **Techniques Used**: Data visualization, supervised machine learning (classification), model evaluation.  
- **Outcome**: Achieved high accuracy in predicting flower species with ML models.  

---

### 3. Movie Rating Prediction  
- **Objective**: Predict the rating of a movie based on features such as genre, director, and actors.  
- **Techniques Used**: Regression models, feature encoding, exploratory data analysis.  
- **Outcome**: Developed a regression model that estimates movie ratings effectively.
